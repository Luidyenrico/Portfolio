(*07)Theme & Language System

Dark / Light Mode

CSS custom properties on :root and [data-theme="dark"] drive all color changes. The toggle writes to localStorage and sets data-theme on <html>.

Current theme: Light 
html[data-theme="light"] { --bg-primary: #EAEAE5; }
html[data-theme="dark"]  { --bg-primary: #050a14; }

// JS
const saved = localStorage.getItem('theme') || 'light';
document.documentElement.setAttribute('data-theme', saved);

Language Toggle (PT/EN)

Elements carry data-i18n="key" attributes. A JS dictionary maps keys to translations. On toggle, all matching elements have textContent updated instantly.

Current language: EN 
// HTML
<span data-i18n="hero_title">Design System</span>

// JS
const translations = {
  en: { hero_title: "Design System" },
  pt: { hero_title: "Sistema de Design" }
};
